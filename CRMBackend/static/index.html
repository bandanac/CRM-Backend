Following are all the CRUD operations included

Get all customers

  • [GET] http://localhost:3000/customer

    • - Display all the customers.

Get a single customer through id

  • [GET] http://localhost:3000/customer/1

    • - Display customer having ID 1.

  • [GET] http://localhost:3000/customer/11

    • - Display an error as user doesn't exists.
