Following are all the CRUD operations included

Get all customers

  • [GET] http://localhost:3000/customer

    • - Display all the customers.

Get a single customer through id

  • [GET] http://localhost:3000/customer/8f1f4cb1-aff0-4094-9398-bbf2dc89d6ad

    • - Display customer having ID 8f1f4cb1-aff0-4094-9398-bbf2dc89d6ad.

  • [GET] http://localhost:3000/customer/8f1f4cb1-aff0-4094-9398-bbf2dc89d600

    • - Display an error as user doesn't exists.

Add a customer

  • [POST] http://localhost:3000/customer

    • { "name": "Dahlia Martin", "role": "Mobile Developer", "email": "dahlia_martin@xyz.com", "phone": "+49-6858-4638667", "contacted": false }

    - Add the customer.

Update a customer

  • [PUT] http://localhost:3000/customer/8f1f4cb1-aff0-4094-9398-bbf2dc89d6ad

    • {"8f1f4cb1-aff0-4094-9398-bbf2dc89d6ad": { "id": "8f1f4cb1-aff0-4094-9398-bbf2dc89d6ad", "name": "Aria Bell", "role": "Mobile Developer", "email": "aria_bell@xyz.com", "phone": "+49-7001-3206111", "contacted": false }}

    - Display the updated customer having ID 8f1f4cb1-aff0-4094-9398-bbf2dc89d6ad.

  • [PUT] http://localhost:3000/customer/8f1f4cb1-aff0-4094-9398-bbf2dc89d600

    • - Display an error as user doesn't exists.

Delete a customer

  • [DELETE] http://localhost:3000/customer/8f1f4cb1-aff0-4094-9398-bbf2dc89d6ad

    • - Delete the customer having ID 8f1f4cb1-aff0-4094-9398-bbf2dc89d6ad

  • [DELETE] http://localhost:3000/customer/8f1f4cb1-aff0-4094-9398-bbf2dc89d600

    • - Display an error as user doesn't exists.
